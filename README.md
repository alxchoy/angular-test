# Test Project

This is a test project elaborated for Michael Page Consulting.

## Installation and Run

Consider to use node v12+

```bash
  npm install
  ng serve
```

## Testing

I used default angular libraries for testing like Karma and Jasmine.

```bash
  ng test # to open Karma dashboard and review test
  ng test --code-coverage # same ng test, but create a report in coverage/index.html
```

## Libraries List

- Angular 12.0.1
- Angular Material 12.0.2
